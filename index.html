What is this?

mSearch is a webapp designed to make it easier for people to search for music online. When a user types a query into the searchbox and hits Search, a google search is performed in the background. mSearch simply goes through the results from the google search and displays them in a simple and easy to use way.

How does it work?

There are a lot of hosting sites out there that host a lot of music. Most of them don't provide a search to search through the hosted content so one must have a direct link to the mp3 file to download it. Google indexes these sites like any other and by typing something like: site:zippyshare.com Lady Gaga into a google search, one can search a hosting site like any other. This is what mSearch does.

What are Iterations?

When a user performs a Google search he gets 10 results per page and to get more he needs to advance to the next page of search results. Since mSearch performs Google searches when searching for music it only gets 10 results like a normal Google user. Iterations determines how often mSearch "goes to the next page" in the google results. The more iterations, the more search results.
